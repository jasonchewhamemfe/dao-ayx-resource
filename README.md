# dao-ayx-resource

本仓库用于归档和发布多个独立的 HTML 页面。页面内容涵盖多种场景，不针对任何特定域名或网站。

## 项目简介

该仓库是一个静态资源归档项目，主要存放独立 HTML 文件及其相关资源。每个页面均为独立单元，不依赖外部服务或数据库，适合离线查看或直接部署至静态文件服务器。

## 目录说明

```
dao-ayx-resource/
├── pages/          # 存放各 HTML 页面及其附属资源
├── assets/         # 通用静态资源（如样式、脚本、图片等）
├── index.html      # 首页或导航页（可选）
└── README.md       # 本文件
```

- **pages/**：每个子目录或文件对应一个独立页面，命名规则为页面功能或主题的简短描述。
- **assets/**：存放多个页面可能共用的 CSS、JavaScript、字体、图片等资源。
- **index.html**：如存在，则作为仓库根目录下的默认入口页面，可提供页面导航链接。

## 页面归档说明

- 所有页面均为纯前端实现，无需后端支持。
- 页面之间互不依赖，可以单独访问。
- 页面资源（如图片、样式、脚本）尽量内联或相对路径引用，以保证离线可用性。
- 归档内容可能随时间更新或移除，不保证长期一致性。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时会提交到对应目录。
- 欢迎通过 Issue 提交建议或报告问题，但请注意本仓库不针对特定网站或服务。
- 如需引用本仓库中的页面，请注明来源仓库地址。
